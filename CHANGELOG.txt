v1.8.21 - Tester Bug Fixes Round 2
Date: May 2026

  More fixes from tester feedback on top of the v1.8.13 batch.

  TALENT FIXES
  - Weakest Link (Overdogs): completely wrong description. Now correctly reads
    "Amplifies Weapon Damage by 30% to the lowest ranking enemies in combat"
    with the full tier hierarchy list
  - Memento Kill Confirmed: had an extra Weapon Damage modifier that shouldn't
    be there. Now only applies Skill Efficiency
  - Vile Toxic Delivery: description updated to match exact in-game wording
  - Sawyer's Rooted: was showing a "while in cover" mechanic, now correctly
    shows "+3% WD per second not moving, stacks up to 10x"
  - Quick Draw: stack rate corrected to every 0.3s (was 0.5s)
  - Dragon's Glare: cooldown corrected to 35s (was 40s)
  - NinjaBike Kneepads Parkour!: added "+25% bonus armor for 5s" to
    description, removed incorrect Weapon Damage modifier
  - Tinkerer Brief: removed incorrect Weapon Damage modifier
  - Investor Slotted: added missing Armor Regen modifier

  EXOTIC ITEM FIXES
  - All exotic fixed attributes are now read-only (non-editable) as they
    should be. Affected items: Coyote, Catharsis, Catalyst, Tinkerer,
    Ridgeway's Pride, Collector, Beacon, Provocateur, Birdie's, Bloody
    Knuckles, Rugged Gauntlets, Exodus, Overdogs
  - Beacon core corrected to Weapon Damage (red) - was Skill Tier
  - Provocateur core corrected to Armor (blue) - was Weapon Damage
  - Shocker Punch core corrected to Armor (blue) - was Skill Tier
  - Acosta's Kneepads now has correct fixed attributes (Hazard Protection
    10%, Status Effects 10%)

  GEAR PICKER FIX
  - Named items that belong to a brand (like Providence Defense Named Mask)
    were showing up in both Brand Sets and Named & Exotic groups. They now
    only appear under Brand Sets where they belong

  ACTIVE BONUSES
  - Imminence Armaments 2pc now shows "Increased Threat" instead of the
    raw internal stat key. Expanded the stat label map to cover all
    brand and set bonus types properly

  VILE DOT COUNTER
  - Added a live "Vile DoT / tick" readout in the stat panel when the
    Vile mask is equipped. Shows calculated damage per tick based on your
    current weapon damage, explosive damage, and status effect stats

  TALENT DESCRIPTIONS IN DROPDOWNS
  - Selecting a talent in the standard UI now shows the full description
    below the dropdown. Browsers don't show tooltips on dropdown options
    so this was the only way to surface the info

v1.7.9 — Saint Theme, Icon Infrastructure & QoL Fixes
Date: April 2026

--------------------------------------------------------------------------------
  SAINT THEME — NEW DARK GOTHIC THEME
--------------------------------------------------------------------------------

  A new "Saint" theme has been added, selectable from the Themes dropdown.
  Inspired by the SAINT display typeface by echodesign (2026).

  Visual features:
  - SAINT display font applied to all headers, card names, section labels,
    nav tabs, and modal titles
  - Animated canvas background with multiple layers:
      * Voronoi stained-glass cells — 60 irregular polygon panes in amber,
        blood-red, deep violet and near-black with dark lead lines between them
      * Film grain — 8 pre-baked 256x256 noise tiles cycled randomly each
        frame (zero per-frame pixel writes for performance)
      * Vertical scratches — 18-40 randomised light-leak and dark scratches
        per session
      * Film burns — warm overexposed blobs at corners and edges
      * Dust and speck layer — 280-480 random particles and streaks
      * Rising ember particles — 80 particles weighted toward left side,
        sparse across full width
      * Gothic decorative artwork — large cross, ornate medallion, dagger,
        chain, filigree vine, crack lines, gothic arch, star markers, ghost
        cross watermark — all drawn as pure canvas paths (no images)
      * Dense technical annotation overlay — coordinate clusters, Python
        code blocks, status headers, crosshairs, dashed scan-lines, dot
        markers matching the editorial aesthetic of the font designs
      * Animated film grain — pool of 8 unique tiles cycled with random
        offset each frame, two blend-mode passes (overlay + screen)
      * Final edge burn — hard corner darkening like old film stock
  - Monochrome white accent system (replaces orange)
  - Zero border-radius everywhere (sharp edges throughout)
  - All pages (Spec, SHD, Compare, Planner, Workbench, Library, Prototype)
    fully styled with dark semi-transparent surfaces
  - Tooltips, IO modal, pickers, selects all themed correctly
